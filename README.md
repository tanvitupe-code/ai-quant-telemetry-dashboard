# AI Quant Telemetry Dashboard 🚀

A real-time, state-driven financial telemetry workspace featuring custom semantic asset query parsing and interactive wave charts. Built with a futuristic cyberpunk UI for advanced quantitative analytics.

---

## 🛠️ Tech Stack & Architecture
* **Frontend Core:** React 18, TypeScript, Vite
* **Data Visualization:** Recharts (Dynamic statistical mapping & custom wave charts)
* **Styling:** Tailwind CSS (Cyberpunk neon theme matrix)
* **Icons:** Lucide React

---

## ✨ Key Features
* **Semantic Query Processing:** Dynamic client-side parsing of financial token and asset inputs.
* **Live Telemetry Streams:** Interactive state-driven data graphs with responsive scaling.
* **Responsive Architecture:** Fully modular component layout tailored for heavy data presentations.
